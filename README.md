# AWS Cloud Bootcamp

This repository is dedicated to the Cloud Bootcamp taught by [Jornada de Dados](https://suajornadadedados.com.br). The official repository can be found here: [Official Repository](https://github.com/lvgalvao/data-engineering-roadmap/tree/main/Bootcamp%20-%20Cloud%20para%20dados).

My goal is to build a strong foundation in AWS cloud computing by completing hands-on projects. This bootcamp journey helps me overcome the challenges of cloud implementation by guiding me from the basics to deploying real-world applications.

## Overview

- **Start from Scratch:** I began with little to no prior cloud experience, and this bootcamp guided me step by step from the fundamentals to actual deployments.
- **Comprehensive Learning:** From securing networks and configuring IAM to deploying containerized applications and using serverless functions, this bootcamp covers all essentials.
- **Real-World Projects:** Each module builds towards projects that integrate various AWS services, solidifying my practical knowledge and boosting confidence in handling production-level challenges.

## Getting Started

### Prerequisites

- An [AWS account](https://aws.amazon.com/)
- Basic knowledge of cloud computing and familiarity with command line interfaces
- [Terraform](https://www.terraform.io/) (if working with infrastructure as code)
- Docker (for container-based projects)
- Other tools and SDKs relevant to each project (as specified in the individual project documentation)

### Installation

Clone this repository to your local machine:

```bash
git git@github.com:BrunoChiconato/aws-bootcamp.git
cd aws-bootcamp
```

Follow the instructions provided in each project folder to set up your environment and deploy your projects on AWS.
